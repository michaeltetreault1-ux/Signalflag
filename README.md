# Signal Flag Outdoors Website

Static website for Signal Flag Outdoors, built for GitHub Pages and the domain `signalflagoutdoors.com`.

## Pages

- `index.html`
- `lessons.html`
- `community.html`
- `about.html`
- `safety-waiver.html`
- `contact.html`

## GitHub Pages Setup

1. Create or open the GitHub repository named `Signalflag`.
2. Upload the contents of this folder to the repository root.
3. In GitHub, go to Settings > Pages.
4. Set the source to the main branch and root folder.
5. Add the custom domain `signalflagoutdoors.com`.

## Domain DNS

For an apex domain on GitHub Pages, point `signalflagoutdoors.com` to GitHub Pages using GitHub's current DNS instructions. Add a `www` CNAME if you want `www.signalflagoutdoors.com` to work too.

## Contact Form

The contact form is static and opens an email draft to `hello@signalflagoutdoors.com`. Replace that email in `contact.html`, `index.html`, and footer links if needed.
